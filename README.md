# Flower Watering Optimization - Algorithm Solution

## 📝 Problem Description
This repository contains an optimized C++ solution for a resource distribution problem. The scenario models a system where $N$ entities (flowers) have specific capacities and current states, and a limited resource $M$ (water) must be allocated efficiently based on specific optimization constraints.

## 🚀 Technical Highlights
* **Language:** C++17
* **Paradigm:** Greedy Algorithm / Resource Allocation
* **Optimization (Fast I/O):** Utilizes `ios_base::sync_with_stdio(false); cin.tie(nullptr);` to optimize standard input/output streams, making it suitable for competitive programming and large datasets ($O(N)$ time complexity).
